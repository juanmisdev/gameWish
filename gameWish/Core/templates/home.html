¿Qué es GameWish?

GameWish es una aplicación web que te permite crear una lista de deseos de videojuegos.

Además, puedes compartir tu lista de deseos con tus amigos, para que sepan qué videojuegos te gustaría tener.

Y mucho más!!!

¿Cómo funciona?

Para empezar a usar GameWish, tienes que registrarte.

Registrarse

Una vez registrado, puedes añadir videojuegos a tu lista de deseos buscandolos en la barra de navegacion.
